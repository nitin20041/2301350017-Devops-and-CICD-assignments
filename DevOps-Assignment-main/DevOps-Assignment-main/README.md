# DevOps Assignments

A collection of hands-on DevOps assignments covering Docker, CI/CD pipelines, infrastructure monitoring, and cloud provisioning with Terraform.

---

## Contents

### Assignment 1 — DevOps Practical Work
A Flask web application containerised with Docker, tested with pytest, and deployed via GitHub Actions to GitHub Container Registry. Includes Redis as a sidecar service and Terraform scripts to provision an AWS EC2 instance.

**Key concepts:** Docker, Docker Compose, Redis, GitHub Actions CI, Terraform, AWS EC2

### Assignment 2 — Docker Infrastructure
A production-grade FastAPI To-Do API with a multi-stage Docker build, SQLite database, structured JSON logging, and Prometheus metrics. Features a full CI + CD pipeline that automatically pushes images to GHCR on merge.

**Key concepts:** FastAPI, Multi-stage Docker builds, Prometheus, GitHub Actions CI/CD, GHCR

---

## Topics Covered

- Containerisation with Docker and Docker Compose
- CI/CD pipelines with GitHub Actions
- Infrastructure as Code with Terraform (AWS)
- Application observability — logging and metrics
- Security best practices (non-root containers, secrets management)
